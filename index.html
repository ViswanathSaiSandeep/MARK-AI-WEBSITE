ABOUT PARIS AI



POWERED BY GOOGLE GEMINI
PARIS AI is integrated by Google Gemini API Key which uses Gemini 2.5 Flash model for answering queries, image generation and answering queries through audio, video, screen sharing features. The data is not stored in any database it is safe secure once the application is closed the data will be deleted permanently.

DESCRIPTION OF PARIS AI
PARIS AI is a voice-controlled desktop assistant for Windows PCs, developed using Python and Tkinter. It features integration with Google Gemini API , enabling smart & accurate responses.

The GUI is simple & user-friendly, designed for every user. PARIS AI responds to voice commands & performs tasks efficiently, offering a smooth user experience.

The application size is around 1.5 GB due to various dependencies. This project showcases skills in Python, API integration, & desktop UI design.

FEATURES OF PARIS AI


INTERACTIVE GUI
An User-Friendly Welcome Screen of PARIS AI with 2 interactive buttons to start and stop the assistant.

GEMINI INTEGRATION
Gemini AI is integrated deeply in PARIS AI to answer the queries within a less time. It summarises the answer and provides the short and simple answer.
Differentiating the User input & the AI output for identification.


IMAGE GENERATION
For image generation it uses Google Gemini Image Generation Model for High quality Image generation which can be used for free for limited usage.
